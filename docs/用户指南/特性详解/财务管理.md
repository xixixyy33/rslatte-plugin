# 财务管理

## 概述

**财务**侧栏视图类型为 `rslatte-financepanel`（`FinanceSidePanelView`）。数据来自**财务索引**（扫描日记中的流水）与**分析快照**（按月生成的统计与规则告警）。界面分两大页签：**财务记录清单**（流水 + 饼图 + 筛选）与 **财务统计明细**（汇总、数据池、衍生指标、告警）。

## 如何打开

- **命令**：`rslatte-open-finance-panel`（命令面板：**「打开侧边栏：财务」**）

模块未在设置中启用时，侧栏仅提示「财务模块未启用」。

---

## 期间与主结构

### 月份 / 年份

- **月份**、**年份** 两个按钮切换视图粒度。  
- **月份** 模式：用 **`type="month"`** 的「选择月份」框选 `YYYY-MM`。  
- **年份** 模式：用数字输入框选 **年份**；统计会按该年 12 个月聚合（能命中几个月快照会显示在汇总区）。

### 两个主 Tab

| Tab | 作用 |
|-----|------|
| **财务记录清单** | 按日时间轴浏览流水；支持分类/子分类/收支筛选；可选支出饼图。 |
| **财务统计明细** | 期间 KPI、数据池汇总、衍生指标、分析索引告警、业务规则告警等。 |

---

## 标题栏按钮（图标）

标题 **「💰 财务」** 右侧为一组操作（实现见 `FinanceSidePanelView` 标题行）：

| 图标 | 含义 |
|------|------|
| **➕** | **新增财务记录**。打开录入弹窗；保存成功后会刷新当前侧栏。 |
| **⚠** | **财务异常清单**。打开弹窗，用于查看 **无 meta**、**同文件或跨文件重复 entry_id** 等异常（便于修数据后再重建索引）。 |
| **🧱** | **扫描重建财务索引（全量）**。重扫日记重建索引，耗时长于增量刷新。 |
| **🗄** | **索引归档（财务）**。将**超过阈值**的记录从**主索引**迁入 **archive**；日记里的笔记**不会**被移动。 |
| **🔄** | **财务手动刷新（增量）**。在阈值范围内扫描有变更的日记，更新索引。 |

其中 **🧱 / 🗄 / 🔄** 受设置 **`uiHeaderButtons.finance`** 控制：可在配置里单独关闭某一类（**➕**、**⚠** 不受此项影响，始终可用）。默认三者均为开启。

## 「财务记录清单」页签

### 支出饼图（可选）

- 当设置 **`rslattePanelShowFinancePieCharts`** 不为 `false`（默认即展示）且当前期间存在**支出**流水时，清单上方会渲染 **按分类的支出饼图**。  
- 仅统计 **未标记删除** 的支出记录。

### 清单标题行

- 展示当前筛选结果下的 **支出合计** 与 **收入合计**（与下方时间轴一致）。

### 筛选条

| 控件 | 作用 |
|------|------|
| **财务分类** | 下拉按设置中的 `financeCategories` 过滤；选「全部」不按分类过滤。 |
| **财务子分类** | 依赖已选分类；从该分类的 `subCategories` 列出选项。未选主分类时子分类下拉为禁用。匹配时从笔记 `note` 中解析子分类（与索引字段一致）。 |
| **类型** | **全部 / 支出 / 收入**。 |

### 时间轴每一行

- **左侧圆点**：支出为 💰，收入为 💵；已逻辑取消的记录显示 ❌。  
- **点击行主体**（避开按钮）：在编辑器中**打开对应日记**，并**滚动到该条流水主行**（优先用索引里的 `sourceFilePath` + `sourceLineMain`，否则在当日日记中按 entry_id / 金额等反查）。  
- **❌ 按钮**（未删除记录才有）：**取消此财务记录**——在日记中写入带 `is_delete` 的对应流水（需有 **entry_id**；若无会提示先重建索引或使用带 meta 的流程记账）。

从其他入口 **跳转并高亮** 某条流水时，侧栏会切到本页签、按账单月份筛选，并对匹配 `entry_id` 的行做短暂高亮（`applyLedgerNavFocus`）。

## 「财务统计明细」页签

本页标题为 **「财务统计明细」**，内容自上而下为多块 **统计块**（`rslatte-finance-stats-block`）。以下按阅读顺序说明。

### 1. 分析快照操作条（仅「月份」视图）

在 **年份** 视图下会提示：请先切回 **月份** 再使用快照按钮。

| 按钮 | 作用 |
|------|------|
| **🔄 刷新当前年月分析快照** | 按**当前索引**重算所选 **`YYYY-MM`** 月的 **stats** 与 **规则告警** 快照，写入 `finance-analysis/snapshots/month/`。写入前会做**轮转备份**（主文件 + `bak1` + `bak2`，与 Review 侧一致，至多保留 3 版）。 |
| **↩ 回退快照版本** | 将该月 **stats**、**alerts** 各**回退一档**（依赖已存在 `.bak1.json` 链）；若从未成功刷新过则可能无可回退。 |

未生成快照时，部分汇总会 **回退为现算结果**（界面上会注明读取源或提示未命中 snapshot）。

### 2. 统计汇总（所选期间）

- **标题**：`统计汇总（期间标签）`。  
- **元信息**：  
  - **月视图**：若存在当月快照，会显示 snapshot 的 `periodKey` 与 `generatedAt`。  
  - **年视图**：说明为**按月快照聚合**，并显示 **命中几个月 / 12**。  
  - 若未命中当期 snapshot 但存在全局 `analysisIndex.latest`，会提示当前为 **现算结果** 并标明 `latest` 月份。  
- **指标行（KV）**：  
  - **有效笔数**  
  - **总收入** / **总支出**  
  - **结余（收入−支出）**  
  - **预算使用率**：仅当 **月视图** 且快照里提供 `budgetUsageRatio` 时显示（百分比）。

月/年的总收入支结余：优先使用 **各月 stats 快照** 的 `summary` 聚合；无快照时退回为当前索引在界面已过滤的 `periodRecords` **现算**。

### 3. 数据池汇总（本期间）

- 展示 **数据池（pool）** 维度金额，名称优先来自空间索引目录下 **`finance-config/finance-data-pools.json`** 中的池名称映射。  
- 列表为金额 **前 12 名** 的正向池（按 snapshot 的 `poolStats`；年视图为 12 个月池金额合并）。  
- 说明文字注明：该块按**数据池口径**统计，用于承接 **规则与预算分析**。  
- 若无 snapshot，会提示 **回退到现算分类统计**（与池块逻辑不同，勿混读）。

### 4. 衍生指标（本期间）

来自快照中的 **`derivedMetrics`**（年视图在无快照字段时用当年 KPI **推导**缺口与结余率）：

- **现金流缺口（支出−收入）**  
- **结余率**（有收入时）  
- 若快照中存在：**自由支出占收入比**、**刚需支出占收入比** 等，会一并列出。

无数据时提示「未命中 snapshot 衍生指标」。

### 5. 分析基础告警（财务分析索引）

- 依赖文件 **`finance-analysis.alert-index.json`**。若不存在，提示先执行财务 **刷新** 或 **重建索引**。  
- 若存在：展示 **索引状态、生成时间、缺失项数量**；下列出 **「基础数据缺失」** 类条目（标题、详情、可选提示）。  
- **期间门闩**：若当前所选月/年 **有效笔数为 0**，会明确提示「当前月/年缺少财务流水」，避免误以为「无告警即正常」。

### 6. 业务告警（规则 JSON 分析结果）

- **月视图**以 `finance-analysis/snapshots/month/{YYYY-MM}.alerts.json` 为准（**不会**误用全局 `finance-rules.alerts.json` 的「当前自然月」，以免选 2 月却看到 3 月数据）。  
- **年视图**：聚合 12 个月的 alerts 快照，并可能额外展示 **期间动态空池告警**（扫描已启用规则依赖的数据池，在当前期间金额为 0 时生成）。  
- **筛选条**：  
  - **严重度**：全部 / high / warning / notice  
  - **状态**：全部 / new / ongoing / resolved / ignored  
  - **分组**：由规则 ID 前缀解析出的规则分组（如 `RULE_XXX_...` 中的段）  
  - **已隐藏告警管理（N）**：打开弹窗，查看被 **短期隐藏** 的告警；可单条恢复或 **全部恢复**。  
- **单条告警操作**：  
  - **关联 N 条**：当告警带有 `relatedEntryIds` 时，打开关联记录列表，可选中后在日记中打开对应流水。  
  - **确认**：**短期隐藏该告警 7 天**（按指纹或 ruleId+周期+文案生成键写入隐藏表）。  
- **月视图列表分页**：每页 **12 条**，底部 **<< / 上一页 / 下一页 / >>** 与页码。  
- **年视图**：可按月折叠（`<details>`），每月默认展示前 **6** 条；动态空池类另有一块摘要。

若所选月 **尚无** alerts 快照，界面会说明需先 **刷新/重建** 以生成该月文件；此时仍可能只看到 **期间动态空池** 类合成告警。

## 与其他能力的关系

- **今日打卡侧栏**内常有财务快捷入口，与这里是同一套索引体系。  
- 插件启动时会尝试拉取财务相关汇总（日记统计）；路径或格式不对时，清单与统计可能为空或滞后，优先检查 **日记目录** 与 **财务刷新/重建**。
