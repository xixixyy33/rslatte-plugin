# RSLatte 特性详解

本目录按**功能模块**拆成多篇说明，便于在使用中查阅；与 [《用户手册》](../用户手册.md) 互补——手册侧重首次上手与总览，此处侧重**入口、视图类型与模块关系**。**[用户手册 · 主要特性](../用户手册.md#主要特性)** 中的条目顺序与本页「文档索引」表一致，便于对照。

内容依据当前插件代码中的视图注册、命令 ID 与用户可见文案整理；界面细节以你安装的版本为准。
