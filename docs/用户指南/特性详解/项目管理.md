# 项目管理

## 概述

**项目管理**侧栏用于浏览 **项目文件夹**、**项目任务清单**（`项目任务清单.md`）、**里程碑** 与 **进度图**，并执行新增项目、延期、完成/取消/待归档、与 **输出/日程/联系人** 等联动操作。

- 视图类型：**`rslatte-projectpanel`**  
- 实现类：**`ProjectSidePanelView`**（`src/ui/views/ProjectSidePanelView.ts`）  
- 数据服务：**`projectMgr`**（`ProjectManager`），索引由 **`projectRSLatte`** 维护，主索引文件为当前空间 **`index/project-index.json`**（及 `archive/`、`project-archive-map.json`、队列路径等，见 `src/projectRSLatte/indexStore.ts`）。

**模块开关**：使用 **`isPipelineModuleEnabled("project")`** 与 Pipeline 一致；**关闭时侧栏整页不渲染**（无占位文案）。

## 如何打开

- **命令**：`rslatte-open-project-panel`（**「打开侧边栏：项目」**）  
- **Hub**：工作流 **「项目」** → **`activateWorkflowView("projects")`**  
- **URI**：`obsidian://rslatte-open?module=项目管理`（及 `moduleMap` 中 `project` 等别名）  
- **Obsidian 协议**：`obsidian://rslatte-project?id=...` — 当前实现为 **打开侧栏**；**按 id 自动定位** 在 `main.ts` 中仍为 TODO。  

关闭侧栏（视图卸载）时，会将 **`projectPanel.mainTab`** 重置为 **`list`**，下次打开默认回到 **项目清单**（同一会话内仅折叠侧栏则不一定卸载，仍可能保留上次页签）。

## 打开与首屏性能（SWR）

若 Pipeline 中项目模块已启用、且 **`ensureReady` 尚未完成**：

1. 可尝试读取 **`project-panel-hydrate.json`** 快照 **先渲染列表**，并 Notice「已显示缓存的项目列表，后台同步中…」。  
2. 随后在后台 **`ensureReady` → `refreshDirty` → `writePanelHydrateSnapshot`**；失败则提示手动 🔄。  

否则 **阻塞式 `ensureReady`** 后再渲染。

## 顶栏操作

- **标题**：🎯 项目管理 + **项目 DB 同步状态灯**（与 `getDbSyncIndicator("project")` 一致）。  
- **➕**：**`AddProjectModal`** — 在配置的 **`projectRootDir`** 下新建项目结构。  
- **🧱**：**`runE2(..., "project", "rebuild")`** — 全量重建项目索引。  
- **🗄**：**`manual_archive`** — **两步一次完成**：① 将满足条件的 **项目文件夹** 移到 **`projectArchiveDir`**；② 将主索引条目迁到 **archive 分片**（按钮 `title` 原文）。  
- **🔄**：**`manual_refresh`** — 增量刷新；成功后 **best-effort** **`writeTodayProjectProgressToJournal(true)`** 写当日日记中的项目进度摘要。  

🧱/🗄/🔄 是否显示由 **UI 头按钮可见性** 控制。

## 主区页签（`projectPanel.mainTab`）

| 页签 | 值 | 内容概要 |
|------|-----|----------|
| **项目清单** | `list` | 多区块列表 + 卡片上的推进摘要、打开任务清单等 |
| **项目进度管理** | `progress` | 项目搜索/选择器、选中项目的 **里程碑/任务清单** 工具栏、**项目进度图**（甘特式时间轴）、延期/里程碑/存档类操作入口等 |

切换页签会 **`saveSettings()`**，下次打开仍记住（除非因 **onClose** 被重置为 `list`）。

### 项目清单（`list`）

折叠状态多 **持久化** 在 **`projectPanel`**（如 `inProgressListCollapsed`、`doneListCollapsed` 等）：

1. **进行中的项目** — 默认 **展开**；卡片含名称、状态文案、风险/日期类 meta、**下一步/里程碑推进** 摘要行；点击标题可打开 **`项目任务清单.md`**。复杂操作（延期、里程碑、分析图、存档、完成/取消等）入口在 **项目进度管理**（见代码注释）。  
2. **已完成的项目** — 默认 **折叠**；`renderDoneProject`。  
3. **待归档项目** — 已标记待归档、超阈值后将随归档流程移动（说明见区块标题注释）。  
4. **取消项目清单** — 默认 **折叠**。  

清单卡片带 **`data-project-id` / `data-project-folder-path`**，供 **`scrollToProject`** 在 DOM 中定位。

**任务清单折叠块**（在清单卡片内）：可展开 **按里程碑分组** 的未闭环任务（每里程碑下最多展示前 **30** 条未闭环任务）；里程碑来自任务清单中的 **`#` 标题** 解析。

### 项目进度管理（`progress`）

- **项目里程碑/任务清单**：**全部展开/收起**、**状态筛选**（TODO/IN_PROGRESS/DONE/CANCELLED）、**任务标签筛选**（与 `TASK_TAG_KEYS` 及 `next_action` 等对齐）、**显示条数上限**（设置项控制）。  
- **项目进度图**：甘特式 **时间轴 + 里程碑竖线 + 任务条**；可 **隐藏/显示已完成任务**；范围与 **左右余量天数** 等由 **`buildProgressChartModel`** 与设置项决定（见 `projectProgressChart.ts`）。  
- 详情区还提供与 **输出、日程、编辑项目/任务** 等相关的模态（具体以当前 UI 为准）。

## 外部跳转：`scrollToProject`

**`scrollToProject(projectIdOrFolderPath, milestonePath?, taskFilePath?, taskLineNo?, navOpts?)`**

- **解析顺序**：**优先按 `folderPath` 匹配，再按 `projectId`**，避免 id 与路径字符串冲突时误命中（见 **`resolveProjectEntryByIdOrFolderPath`** 注释）。  
- 若需定位 **具体项目任务行**，会自动切到 **`progress`** 页签、设置 **`progressSelectedProjectId`**，并可由 **`navOpts`** 写入 **标签筛选 / 保证某状态可见**（供 Today 等入口收窄列表）。  
- 任务行 DOM 使用 **`data-project-task-file-path`**、**`data-project-task-line-no`**（与联系人侧栏跳转项目任务一致）。  

**联系人 / Today / 日历** 等通过 **`activateProjectView` + `scrollToProject`**（或等价路径）打开本侧栏并滚动高亮。

## 索引与空间

- **`project-index.json`**、**`project-archive-*.json`**、**`project-archive-map.json`** 等位于 **当前空间的中央 `index/`**（与 [空间与索引](空间与索引.md) 一致）；**`projectRootDir` / `projectArchiveDir`** 等属 **空间级设置快照**。  
- 切换空间后 **`projectMgr`** 会 **resetStore / clearAllSnapshots**（见 `spaceManagement`），应依赖 **🔄** 或重新打开侧栏加载当前空间数据。

## 与其他模块的关系

- **任务管理**：日记任务与 **项目任务** 不同载体；项目任务行可录 **日程**（`RecordTaskScheduleModal` 等），日程索引与任务侧栏共用。  
- **输出管理**：项目类输出与项目文件夹/模板体系关联（见输出文档与模板合并逻辑）。  
- **联系人**：项目任务描述中的 **`[[C_…]]`** 可进入联系人 **互动索引**。  
- **Today**：今日执行中的 **项目推进** 卡片跳转本侧栏并 **`scrollToProject`**；统计块可 **`openInProgressListFromStats`**（输出侧栏专用，项目为同源导航思路）。
