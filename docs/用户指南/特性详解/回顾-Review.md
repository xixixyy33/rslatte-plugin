# 回顾（Review）

## 概述

**回顾**为 V2 统一复盘侧栏（`ReviewView`，视图类型 **`rslatte-review`**）。以 **周 / 月 / 季** 为周期，在 **当前空间** 下汇总索引与操作日志，分三块阅读：

1. **执行**：任务、提醒、日程、项目、输出、联系人等「做了什么」—含概览条、各模块统计句、日程可视化（周栅格 / 月热力 / 条形图等）、可点击跳转时间轴类条目。  
2. **核对**：基于本周期 **已闭环** 记录的 **及时性**、与 **上一等长周期** 的 **环比**、以及周期事实与简短 **建议**（非 LLM）。  
3. **记录**：**打卡、财务、健康、日记** 四域 — 周期摘要、记录类 WorkEvent 抽样时间轴、分项分析、自动拼句纪要；并支持 **周/月/季报告** 路径说明与 **复制执行摘要**。

设计脉络见 [`docs/V2改造方案/08-Review侧边栏优化方案.md`](../V2改造方案/08-Review侧边栏优化方案.md)（§4.3 / §4.3a）；以下为与代码一致的行为说明。

## 如何打开

- **命令**：`rslatte-open-review-panel`（**「打开侧边栏：回顾」**）  
- **Hub**：工作流 **「回顾」**  
- **URI**：`obsidian://rslatte-open?module=回顾`（及 `Review`、`review` 等映射）

### 深度打开（周期与子页签）

插件 API **`activateReviewView(opts?)`**（`src/plugin/uiNavigation.ts`）可在打开前合并参数，内部使用 **`markPendingReviewOpen`**，下一帧渲染时消费：

| 参数 | 说明 |
|------|------|
| `grain` | `"week"` \| `"month"` \| `"quarter"` |
| `periodKey` | 周：`YYYY-Www`（如 `2026-W13`）；月：`YYYY-MM`；季：`YYYY-Q1`～`Q4` |
| `periodOffset` | 相对**当前**周/月/季的整数偏移（与顶栏「上一/下一周期」同一套） |
| `subTab` | `"execute"` \| `"reconcile"` \| `"records"` |

若同时传 `periodKey` 与 `periodOffset`，以 **`periodOffset` 为准**（当 `periodOffset` 为有限数时）。

示例（脚本或内部调用）：`activateReviewView({ grain: "month", periodKey: "2026-03", subTab: "records" })`。

## 周期工具栏

- **周 / 月 / 季**：切换粒度；切换后 **周期偏移归零**（回到「本周/本月/本季」相对当前日历）。  
- **← 上一周期** / **下一周期 →**：`_periodOffset` 减 1 / 加 1。  
- **周期标题行**：`periodKey`（如 `2026-W13`、`2026-04`、`2026-Q2`）、起止日期 `YYYY-MM-DD`、状态标签：  
  - **进行中**：今天在 `[startYmd, endYmd]` 内；  
  - **已结束**：整个周期落在过去。

### 进行中周期

- 右侧 **🔄**：**重新执行 `render()`**（刷新界面与实时模型），**不**打开快照弹窗。

### 已结束周期

- **🔄**：打开 **`ReviewHistorySnapshotModal`**（**刷新/管理历史快照**）：可查看、保存本周期三页数据的 **打包快照**（与财务分析快照类似，有版本与备份策略；保存成功后在 `onSaved` 里清空缓存并重绘）。  
- **↩**（仅当存在 **手动快照** 时显示）：**回退最新一版手动快照**；确认框说明回退后可能回到上一版或 **实时索引 / E2 封印** 显示。成功后 Notice 提示剩余版数。

快照文件落在 **当前空间根目录** 下的 **`review-snapshots/`**（`reviewSnapshotFilePath`），手动版本与 **E2 自动「已完成」封印**（`*.completed.json`）由 `readReviewSnapshotHead` 等逻辑择优展示。默认最多保留 **`REVIEW_SNAPSHOT_MAX_VERSIONS`（3）** 版手动快照。

### 横幅（已结束周期）

- 若当前读到的 **`snapHead.latest`** 非空：横幅说明展示的是 **E2 自动封印** 还是 **手动快照**（含版数 `manualVersionCount/3`）及保存时间。  
- 若无快照：提示当前为 **实时主索引 + 操作日志** 计算；**季** 还会说明已 **合并归档分片**（任务/提醒/日程/输出）；**周/月** 提醒旧周期可能因 **归档** 导致主索引不完整。

### 季 · 进行中 的补充说明

- 当 **粒度为季** 且周期 **进行中** 时，会显示一条 **灰字说明**：本季统计已 **合并归档索引分片**，与 **周/月** 口径不同。

## 三个子页签（名称随粒度变化）

| `subTab` | 周 | 月 | 季 |
|----------|----|----|-----|
| `execute` | 本周执行 | 本月执行 | 本季执行 |
| `reconcile` | 本周核对 | 本月核对 | 本季核对 |
| `records` | 本周记录 | 本月记录 | 本季记录 |

切换子页签会 **整页重绘**；**已结束周期** 下会缓存当前 `spaceId|grain|periodKey` 的 **快照头**，避免反复读盘。

### 数据从哪来（概要）

- **进行中** 或 **无快照的已结束**：各 `buildReview*Model` 实时读 **当前空间** 的任务/提醒/日程/输出等索引（周月季会按 `reviewIndexMerge` 等规则 **合并归档**），以及 **WorkEvent**（操作日志 JSONL）。  
- **有快照的已结束**：三页分别使用快照内 **`payload.execute` / `reconcile` / `records`**，保证与封存时一致。

加载失败时对应页显示 **「…页数据加载失败，请稍后重试。」**

---

## 「执行」页（execute）

- **总览条**：项目 / 输出 / 联系人等模块的 **周期内完成、发布、新建** 等数字条（带图标分区）。  
- **分模块段落**：**任务**（周期内完成数、新建、晚于计划结束完成、日记估算工时 + 项目任务估算工时等）、**提醒**（完成/新建、生日/纪念日/到期类计数、转任务/转日程等）、**日程**（完成与时长等）、**项目**、**输出**、**联系人** 等 — 具体句式以界面为准。  
- **日程可视化**：例如 **周 · 6:00–22:00 时间栅格**（已完成且有起止时刻的块）、**月 · 日历热力**（按完成日汇总时长）、**条形图** 等；历史快照下可能带「旧版口径」说明。  
- **时间轴/列表**：任务与项目任务等条目可 **点击**，配合 `reviewTimelineNavigate` 打开对应笔记或侧栏（以当前实现为准）。

---

## 「核对」页（reconcile）

- **本页侧重**：说明用索引 **本周期已闭环** 算及时性，用 **完成/新建** 与 **项目完结、操作日志（发布、联系人）** 等做对比；**开放项清单不在本页**。  
- **A 及时性**：任务（有计划结束日的闭环里按时/偏晚、及时率）、日程（当日闭环占比）、提醒（相对提醒日的及时率）— 均为 **客观计数句**。  
- **B 较上一同期**：与 **等长上一周期** 对比完成/新增及日志类增量（无法算上一周期时会说明）。  
- **C 建议**：根据 A/B 与周期事实生成的 **短句建议**；若无则提示自行结合 A/B 复盘。

---

## 「记录」页（records）

- **本页侧重**：仅 **打卡、财务、健康、日记**；任务/提醒/日程等见「执行」，及时性与环比见「核对」。  
- **周期报告**：按粒度说明 **周/月/季** 报告默认落盘路径（相对 **日记路径上一级** 的 `weekly/`、`monthly/`、`quarterly/`），以及设置中的 **子窗口** 文案；提供 **复制执行摘要** 等按钮（以界面为准）。  
- **A 周期记录摘要**：四类计数；**日记「天」** 指至少一个已配置 **日记面板** 内有有效字的日子；芯片可 **跳转对应侧栏**。  
- **B 时间分析**：打卡/财务/健康 的 WorkEvent **抽样**（上限 `timelineSampleCap`），按日聚合、时间倒序；内链显示名折叠与提醒侧一致。  
- **C 记录摘要**：四域分列分析（打卡履行、财务告警、健康达标与规则、日记子类字数环比等）；健康可能带 **月快照** 说明（周期末所在月）。  
- **D 周期纪要**：由 A 区计数 **自动拼句**（非 LLM）。

---

## 使用建议

- **进行中周期**：数据随索引与日记变化；改完任务/输出后可在本侧栏点 **🔄** 或切换子页签触发重绘。  
- **已结束周期**：需要 **稳定留档** 时用历史快照弹窗 **保存**；之后阅读以横幅所示快照为准。  
- 若大面积空白或数字异常：先 **刷新相关模块索引**，确认 **当前空间** 正确，并 **开启工作事件** 以便核对页使用操作日志。
