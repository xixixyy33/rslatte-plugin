# 输出管理

## 概述

**输出管理**侧栏用于在 Vault 内维护 **输出类 Markdown 文档**（模板创建、状态流转、等待恢复、关联日程、发布到知识库与历史检索）。视图类型 **`rslatte-outputpanel`**，实现类 **`OutputSidePanelView`**（`src/ui/views/OutputSidePanelView.ts`）。台账与列表数据来自 **`outputRSLatte`** 服务及中央索引 **`output-index.json`**（按当前空间目录解析，见 [空间与索引](空间与索引.md)）。

## 如何打开

- **命令**：`rslatte-open-output-panel`（**「打开侧边栏：输出」**）  
- **URI**：`obsidian://rslatte-open?module=输出管理`（及 `moduleMap` 中 `output` 等别名）  
- **Obsidian 协议**：`obsidian://rslatte-output?...`（`main.ts` 中注册；当前会对 **打开输出侧栏** 生效，**在侧栏内精确定位某条输出** 仍为 TODO 注释中的后续能力）  
- **程序化**：**`activateOutputView()`**

**外部联动**：**Today「今日执行」统计** 等可调用 **`openInProgressListFromStats()`**，强制切回 **「正在输出」** 页签并 **展开「进行中」清单**；亦可通过 **`focusTimelineItemByFilePath(path)`**（内部写入 **`__rslatteOutputFocusPath`**）在时间轴中高亮指定文件。

## 模块开关

侧栏是否渲染由 **`moduleEnabledV2.output`** 决定（与 **`isPipelineModuleEnabled("output")`** 组合判定的逻辑见 **`render`** 开头）。**关闭时整页不渲染**（无「模块已关闭」占位文案）。

## 顶栏与索引操作

- 标题 **📤 输出管理** + **output 模块 DB 同步状态灯**（若启用）。  
- **➕**：**`AddOutputTemplateModal`** — 新增输出文档模板。  
- **📋**：**`ManageOutputTemplatesModal`** — 管理模板（与设置中模板表一致）。  
- **🧱**：Pipeline **`output` / `rebuild`** — 全量扫描重建 **`output-index.json`**。  
- **🗄**：**`manual_archive`** — 将 **已完成且超阈值** 的输出笔记移入归档根并更新索引（说明见按钮 `title`）。  
- **🔄**：**`manual_refresh`** — 增量刷新索引；若开启 **output DB 同步** 会一并触发同步。  

以上 🧱/🗄/🔄 是否显示受 **UI 头按钮可见性** 配置约束（与其它侧栏一致）。

## 快速创建

「快速创建」区展示 **一般范围** 且标记为 **可快速创建** 的模板（**`isQuickCreateOutputTemplate`**）。  
默认按 **`templateCreateCounts`** 使用频率排序，取 **前 4** 为按钮，其余收入 **「...」** 浮层。点击后打开 **`CreateOutputDocModal`**。

若无可用模板，会提示在设置或「管理模板」中添加 **范围=一般** 且启用的模板。

## 主区页签

由 **`outputPanel.sidePanelMainTab`** 持久化，两个页签：

| 页签 | 设置值 | 说明 |
|------|--------|------|
| **正在输出** | `list`（默认） | 按状态分组的输出时间轴 |
| **历史发布清单** | `knowledge_publish` | 在知识库根下扫描 **已发布** 文档 |

### 正在输出

渲染前会 **`refreshIndexNow({ mode: "full" })`**，保证 **已完成 / 已取消** 等也进入快照；并 **best-effort** 调用 **`resumeWaitingOutputsIfDue`** 处理到期恢复。

- **清单筛查**（持久化）：**一般输出** / **项目输出** / **全部**（与 **`outputIndexItemIsProjectKind`** 一致）。  
- **时间轴排序字段**（**`timelineTimeField`**）：**`mtime`**（默认）/ **`create`** / **`done`**。  
- **每清单最多条数**（**`maxItems`**）：约 **1～50**，默认 **20**。  

**四个可折叠分组**（会话内默认：进行中展开；项目已完成、取消默认收起；一般已完成默认展开）：

1. **进行中** — `todo` / `in-progress` / `waiting_until`  
2. **项目输出已完成** — `done` 且项目类  
3. **一般输出已完成** — `done` 且非项目类  
4. **取消** — `cancelled`  

**单行操作**（与状态有关，节选）：

- 进行中类：**✏️** 修正属性（**`EditOutputMetaModal`**）、**📅** 录日程（**`RecordTaskScheduleModal`**，`kind: "output"`）、**▶/↻/✅/⛔** 等状态切换。  
- **`waiting_until`**：**↻** 设置 **`resume_at`**（**`SetOutputWaitingModal`**）、**▶** 立即恢复进行中、**⛔** 取消。  
- **`done`**：**📚** 发布到知识库（**`PublishToKnowledgeModal`**）；若已在库中存在副本，可显示 **复制记录** 行，点击跳转知识库对应文件。  
- **`done` / `cancelled`**：**♻️** 回到 **`in-progress`**（草稿态）。  

状态变更会 **更新 frontmatter**（含 **`start` / `start_time`、`done` / `done_time`、`cancelled` / `cancelled_time`** 等规则）、**`outputRSLatte.upsertFile`**、可选 **DB 同步**、**日记进度摘要**、**WorkEvent**（`start` / `continued` / `done` / `cancelled` / `paused` / `recover`）及 **`appendOutputStatusChangedLedgerEvent`**。

点击行主体：**在编辑器中打开源文件**（source 模式）。

**关联日程**：若输出关联了日程 UID，会展示可点击行，**跳转任务侧栏 · 日程安排** 并 **`focusScheduleByFileLine`**。

### 历史发布清单

在 **`resolveKnowledgeLibraryRootRel`** 指向的知识库根目录下扫描 Markdown（单次上限 **2000** 篇），仅保留 frontmatter 中 **`published_space_id`** 与 **当前空间 ID** 一致的文件。

- **搜索**：回车提交，匹配路径、标题、**output_id** 等子串。  
- **output_id**：回车提交，**精确匹配**（至多展示 **30** 条）。  
- 默认列表：**当前空间最近发布** 前 **30** 条，按 **`published_at` / `publishedAt` / mtime** 降序。  

每条可 **阅读**（预览模式打开）、**打回输出**（**`RecallOutputFromKnowledgeModal`**）。

## 与知识库、任务的关系

- **发布**：将 **已完成** 输出复制/整理到 **`30-Knowledge`**（或设置中的知识库根）下约定结构，由 **`PublishToKnowledgeModal`** 完成；发布后可在 **知识** 视图浏览（见 [知识库阅览](知识库阅览.md)）。  
- **跳转知识库**：已完成条目下的 **复制记录** 或历史清单中的操作会 **`activateKnowledgeView`** 或聚焦已有 Knowledge 叶，并设置 **`__rslatteKnowledgeFocusPath`** 以刷新定位。  
- **日程**：输出侧栏录入的关联日程与 **任务管理 · 日程安排** 共用 **`schedule-index`** 与跳转逻辑。

## 索引与设置要点

| 产物 / 配置 | 说明 |
|-------------|------|
| **`output-index.json`** | 当前空间索引目录下的主索引 |
| **`output-sync-state.json`** | DB 同步队列与重试状态（若启用同步） |
| **`outputPanel`** | 模板、归档根、列表筛选、时间轴字段、`sidePanelMainTab` 等（**按空间快照** 隔离，见 `spaceSettings`） |

勿随意删除正在使用的索引文件；异常时优先 **🔄 刷新**，必要时 **🧱 重建**。
