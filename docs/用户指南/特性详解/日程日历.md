# 日程日历

## 概述

**日程日历**是独立侧栏视图，类型 **`rslatte-calendar`**，实现类 **`CalendarView`**（`src/ui/views/CalendarView.ts`）。用 **月历格** 展示 **`schedule-index`** 中的日程分布，点选某日后在下方展开 **当日列表 + 24 小时泳道**，并与 **任务侧栏 · 日程安排** 联动跳转。

与 **任务管理**里「📅 日程安排」子页签的关系：**同一套 `schedule-index` 数据**；日历侧重 **按日浏览与可视化时间轴**，侧栏侧重 **分组清单与增删改**（见 [任务与提醒](任务与提醒.md)）。

## 如何打开

- **命令**：`rslatte-open-calendar`（**「打开侧边栏：日程日历」**）  
- **程序化**：**`activateCalendarView()`**（`main.ts` 注释：schedule-index 月历 + 选日展开）  
- **URI**：`obsidian://rslatte-open?module=日程日历`（`moduleMap` 别名以 `main.ts` 为准）  

部分 **工作区布局预设**（如象限布局）会把本视图放在左侧栏区域，便于与笔记并排使用。

## 启用条件与数据

- 使用 **`isPipelineModuleEnabled("schedule")`** 判断；未启用时仅显示 **「日程模块未启用」**（与 Pipeline / 初始化门控一致）。  
- **注意**：任务侧栏是否出现 **「日程安排」** 子页签还受 **`moduleEnabledV2.schedule`**（且默认可跟随 `memo`）影响，与 Pipeline 层开关在极端配置下 **可能不完全同步**；若出现「日历能开、侧栏无日程页签」或相反，以当前设置与门控实际行为为准。  
- 每次渲染前 **`taskRSLatte.ensureReady()`**，再 **`readScheduleIndexItems()`** 读取当前空间下的日程索引条目。  
- 监听 **`rslatte:space-changed`**，切换空间后 **自动重绘**，避免仍显示旧空间日程。

## 界面分区

### 1. 日历头

- 显示 **当前浏览月**（如 `YYYY年 M月`）。  
- **◀ / ▶**：上一月 / 下一月；换月时会 **同步选中日期**（若原选中不在新月，则选本月「任务日今天」或该月 1 号，逻辑见 **`syncSelectedToCurrentMonth`**）。  
- **回到今天**：把浏览月切到 **任务日「今天」** 所在月，并把选中格设为 **`getTaskTodayKey(taskPanel)`**（与任务标签、Today、Capture 等 **「今日」口径一致**，非纯日历日）。

### 2. 月历格

- 标准 **周日～周六** 表头；格内为当月日期数字。  
- **密度点**：当日程条数增多时显示更多小点（**`densityDotsForCount`**），用于扫一眼繁忙程度。  
- **「!」**：当日存在 **时间段重叠**（**`dayHasScheduleTimeOverlap`**）。  
- **高亮**：**任务日「今天」** 与 **当前选中日期** 有不同样式。  
- 点击某日 → 更新 **选中日期** 并刷新 **第三区**。

### 3. 选中日程展开

标题为 **「选中日程 · YYYY-MM-DD」**。

- **无条目**：显示「当日暂无日程」。  
- **有条目**：  
  - **列表行**：时间摘要（**`formatScheduleTimeSummary`**）、**日程分类**（**`scheduleModule` + `labelForScheduleCategoryId`**）、描述正文（去掉冗余时段前缀后展示；已完成带 **✅**）。正文中的联系人引用会 **解析显示**（**`renderTextWithContactRefsResolved`**）。  
  - **重叠**：与别条时间冲突时行上标 **「叠」**，泳道上亦有重叠区域提示。  
  - **进行中**：若当前时刻落在该条日程时间内，列表行与泳道段可加 **进行中** 样式（**`isScheduleInProgressNow`**）；若选中的是 **任务日今天**，泳道上会画 **当前时刻竖线**（**`now-tick`**）。  
  - **关联图标**（与 Today「今日核对」泳道一致）：**🗂** 关联任务、**🎯** 关联项目任务、**📄** 关联输出（**`resolveScheduleCalendarLinkFlags`**）。  
  - **泳道**：按开始/结束时间排在 **0:00–24:00** 轴上，多段自动 **分 lane**（**`buildScheduleDaySwimlaneSegments`**）；鼠标悬停泳道段会 **高亮对应列表行**。  

**点击某条日程**：调用 **`activateTaskView()`**，再对 **`TaskSidePanelView`** 执行 **`focusScheduleByFileLine(filePath, lineNo)`** — 切到 **「日程安排」** 子页签并滚动高亮对应行；失败则 **打开源笔记**（与索引中的 `filePath` / `lineNo` 一致）。

> 本视图 **没有**单独的 🔄 重建/刷新按钮；索引更新依赖 **任务侧栏日程区** 的手动刷新、Pipeline 自动刷新或全局设置中的自动刷新策略。

## 与任务侧栏的差异小结

| 维度 | 日程日历 | 任务侧栏 · 日程安排 |
|------|----------|---------------------|
| 视图类型 | `rslatte-calendar` | `rslatte-taskpanel` 内子页签 |
| 展示 | 月历 + 选日泳道 | 多分组列表（今日/近期/超期等） |
| 编辑入口 | 无（跳转侧栏或源笔记） | ➕ 新增、行内操作与 Modal |
