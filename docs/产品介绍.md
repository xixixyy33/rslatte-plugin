# RSLatte 产品介绍

**RSLatte**（Obsidian 插件 ID：`rslatte-plugin`）是一款面向 **Obsidian 桌面端** 的扩展，将笔记库扩展为可落地的「生活与工作操作系统」——在熟悉的 Markdown 与文件夹之上，统一管理任务、项目、输出、知识、财务、健康、打卡、联系人与日程等事项，并可选接入后端同步与索引能力。

> **Slogan**：Build Your Life with Flavor —— 在笔记里，把日子过出条理与滋味。

---

## 产品定位

RSLatte 不是单一功能的「小工具合集」，而是一套 **以空间（Vault 分区）与中央索引为骨架** 的插件：把分散在多个 App 里的待办、项目进度、输出稿件、账本与健康记录，收敛到 **同一座知识库** 中，用侧栏、工作台与命令面板串成连贯工作流；需要多端或团队协同时，再通过 **API 同步** 将结构化数据与归档策略延伸到数据库侧。

适合希望 **「笔记即系统」**、又不愿离开 Obsidian 生态的深度用户与小团队。

---

## 你能用它做什么

| 方向 | 能力概要 |
|------|----------|
| **入口与布局** | 工作台（Hub）聚合空间卡片与工作流；可载入内置工作区布局，快速进入常用侧栏与视图。 |
| **当日执行** | 「今天」聚合今日事项；「今日打卡」侧栏覆盖打卡、财务与健康等快捷操作。 |
| **记录与捕获** | Capture 一条输入写入日记或待整理任务；操作日志侧栏呈现工作事件时间轴。 |
| **任务与日程** | 任务与提醒、独立日程日历侧栏；与项目、联系人等模块可联动。 |
| **项目与交付** | 项目管理（里程碑、任务清单、进度）；输出管理（模板、状态、发布到知识库与台账）。 |
| **知识与生活** | 知识库阅览；财务管理、健康管理、打卡统计；联系人管理与互动记录。 |
| **治理与同步** | 多空间与中央索引；配置 API 后的按模块同步；按策略的自动归档。 |

更细的 **命令、视图类型与设置项** 见下文「延伸阅读」中的特性详解与各模块文档。

## 运行环境与版本说明

- **宿主应用**：[Obsidian](https://obsidian.md/)（第三方社区插件）。
- **平台**：当前版本标记为 **仅桌面端**（`isDesktopOnly: true`），请在桌面客户端中使用。
- **最低应用版本**：以 `manifest.json` 中的 `minAppVersion` 为准（当前为 **1.5.0** 及以上建议）。
- **首次使用**：需完成 **插件初始化环境检查** 并在设置中 **完成初始化（启用模块）** 后，各业务模块才会按你的配置生效；详见用户手册对应章节。

## 小结

RSLatte 把 **执行（任务/项目/日程）**、**产出（输出/知识）** 与 **生活数据（财务/健康/打卡/联系人）** 统一在 Obsidian 笔记库内，用工作台与多侧栏降低切换成本；在需要时再通过 **同步与归档** 与后端衔接。若你已经在用 Obsidian 做知识管理，并希望 **少装几个 App、多留一份可检索的本地真相源**，RSLatte 即是围绕这一目标打造的一体化插件方案。
